# CyberSecurityBotGUI

## Project Overview

CyberSecurityBotGUI is a Cybersecurity Awareness Chatbot developed in C# using WPF (Windows Presentation Foundation). The chatbot is designed to educate users about cybersecurity topics while providing interactive features such as task management, quizzes, activity logging, and database integration.

This project was developed as part of a Programming POE and demonstrates the use of object-oriented programming, GUI development, basic NLP simulation, and MySQL database connectivity.

## Features

### Part 2 Features

* Graphical User Interface (GUI) using WPF
* Keyword recognition for cybersecurity topics
* Dynamic chatbot responses
* User memory (name and favourite topic)
* Sentiment detection
* Random cybersecurity tips
* Conversation flow handling
* Cybersecurity information buttons

### Part 3 Features

* Cybersecurity Task Manager
* Task reminders
* Cybersecurity Quiz Game
* Score tracking and feedback
* Activity Log
* Basic NLP simulation using keyword detection
* MySQL database integration for task storage

## Database

The application uses a MySQL database named:

CyberSecurityBot

Table used:

Tasks

Fields:

* Id
* Title
* Description
* Reminder
* Completed

## How to Run

1. Open the project in Visual Studio.
2. Ensure MySQL Server is running.
3. Ensure the CyberSecurityBot database and Tasks table exist.
4. Build and run the application.
5. Use the chatbot interface to interact with the cybersecurity assistant.
